# Fizzbuzz

A simple FizzBuzz implementation on Crystal (study purpose).

## Installation

```sh
$ crystal deps
```

## Tests

```sh
$ crystal spec
```

## Usage

```crystal
Fizzbuzz.of(3) # "Fizz"
Fizzbuzz.of(5) # "Buzz"
Fizzbuzz.of(15) # "FizzBuzz"
Fizzbuzz.of((5..10)) # "Buzz, Fizz, 7, 8, Fizz, Buzz"
```
